# Sreileak Theap — Portfolio

Personal portfolio website built with Next.js, TypeScript, and Tailwind CSS. Showcases my projects, experience, and skills as a Full Stack Software Engineer.

## Tech Stack

- [Next.js](https://nextjs.org/) — React framework with SSR and image optimization
- [TypeScript](https://www.typescriptlang.org/) — Type safety throughout
- [Tailwind CSS](https://tailwindcss.com/) — Utility-first styling
- [Formspree](https://formspree.io/) — Contact form submissions

## Running Locally

**Prerequisites:** Node.js and Yarn installed.

```bash
yarn install
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Customization

All site content lives in `src/data/data.tsx`. Update that file to change personal info, experience, portfolio projects, and contact details.

Portfolio images are in `src/images/portfolio/`.

## Deployment

Deployed on [Vercel](https://vercel.com). Push to `main` to trigger a redeploy.
